# CyberGuard AI

CyberGuard AI is a Flask-based Cloud and IoT Security Risk Assessment Platform that analyzes security configurations and provides risk scores, threat analysis, and actionable recommendations.

## Features

* User Registration and Login
* Cloud and IoT Security Risk Assessment
* Dynamic Risk Score Calculation
* Platform-Specific Threat Detection
* Interactive Threat Analysis Dashboard
* Personalized Security Recommendations
* Assessment History with Delete Option
* SQLite Database Integration

## Supported Platforms

* AWS
* Microsoft Azure
* IoT Devices

## Technology Stack

* Python
* Flask
* HTML
* CSS
* SQLite
* Pandas
* Scikit-learn

## Installation

```bash
git clone https://github.com/Annapurna-hub/CyberGuard-AI.git
cd CyberGuard-AI
pip install -r requirements.txt
python main.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
